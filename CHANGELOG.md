Change Log
==========

## Version 0.5.0

* Repo change: `doubean` -> `doubean-public`
  - New code will only be pushed into the private `doubean`
* Package name change: now `com.github.bumblebee202111.doubean`
* New: Rooted users can synchronize login status from Douban app
  * Manual login with Douban session (its access usually requires root) is available but is not well supported and highly discouraged
* New: Introduce tons of libraries, like Jetpack Compose, Ktor, Kotlinx Serialization, Coil Compose, Compose Material 3, libsu and Daggaer Hilt.
* New: Groups home largely rewritten with these libraries.
